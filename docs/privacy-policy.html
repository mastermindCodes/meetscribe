MeetScribe Privacy Policy Open Source

Last updated: June 11, 2026

Overview

MeetScribe is an open-source Chrome extension for meeting transcription and AI-powered summaries. This policy explains how we handle user data.

Data Collection

What We Collect

  • Meeting captions: Real-time text from Google Meet's built-in captioning feature
  • Speaker names: Extracted from the caption stream
  • Settings: Your preferred AI provider, model choice, and API key

What We Do NOT Collect

  • ✗ Audio recordings — Tactiq does not record or store audio. We only capture text captions.
  • ✗ Video — No video data is captured
  • ✗ Personal information — No names, emails, or contact details beyond what appears in captions
  • ✗ Browsing history — Only Google Meet pages are accessed
  • ✗ Keystrokes or mouse movements

Data Storage

All data remains on your device. Captions and settings are stored using chrome.storage.local, which is encrypted by Chrome and never transmitted to our servers. We operate no backend servers.

Data stored locally:

  • Transcription history (last 50 meetings)
  • Extension settings (AI provider, model, language preference)
  • Your API key (if configured)

AI Processing

When you click "Summarize", "Action Items", or "Ask AI", the transcript text is sent to your chosen AI provider (OpenAI, Anthropic Claude, DeepSeek, or Google Gemini) using your own API key.

  • No data is sent automatically. AI processing is always user-triggered.
  • We do not see your data. The request goes directly from your browser to the AI provider.
  • Your API key is your own. We do not provide or manage AI keys.
  • OpenAI does not use API-submitted data to train models (see OpenAI API Policy).
  • Similar policies apply to Anthropic, DeepSeek, and Google.

Data Sharing

We do not share any user data with third parties. The only data transmission is the transcript you explicitly choose to send to your configured AI provider.

Third-Party Services

MeetScribe integrates with the following third-party services at your direction:

  • OpenAI API (via your key)
  • Anthropic Claude API (via your key)
  • DeepSeek API (via your key)
  • Google Gemini API (via your key)
  • Google Meet (for caption access only)

Each provider has its own privacy policy governing data sent to their API.

Security

  • No external servers operate behind the extension
  • API keys stored in Chrome's encrypted local storage
  • All network requests go directly to providers you select
  • Source code is fully open and auditable on GitHub

Children's Privacy

MeetScribe is not directed at children under 13. We do not knowingly collect personal information from children.

Changes to This Policy
