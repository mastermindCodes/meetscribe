What is MeetScribe?

MeetScribe is a free, open-source Chrome extension that transcribes your Google Meet calls in real-time. No bot joins your meeting — it works entirely through your browser's built-in captions.

📝 Live Transcript

Real-time caption capture in a side panel. Speaker identification with color coding.

🤖 AI Summaries

One-click meeting summaries using OpenAI, Claude, DeepSeek, or Gemini. Your own API key.

✅ Action Items

Automatically extract tasks, owners, and deadlines from conversations.

🌍 Multi-language

Supports 60+ languages via Google Meet captions. AI responds in the same language — including Arabic.

🔒 Privacy First

No audio recording. No bot joins calls. All data stored locally. Open source and auditable.

📤 Export

Export transcripts as TXT, SRT subtitles, or JSON. Share via link or copy to clipboard.

Installation

  1. Clone or download from GitHub
  2. Go to chrome://extensions/
  3. Enable Developer mode
  4. Click Load unpacked and select the meetscribe/ folder
  5. Join a Google Meet call and enable captions (CC button)
  6. Click the MeetScribe icon to open the side panel
